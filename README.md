# Custom PC Build Service - Landing Page

## Description
A professional landing page for a custom PC build service that helps customers design and build their dream gaming or workstation PCs. The service offers personalized consultations, component selection, assembly, and testing.

## Features
- Modern, clean design optimized for conversions
- Responsive navigation
- Service offerings showcase
- PC build gallery
- Contact section with call-to-action
- Pure HTML & CSS (no frameworks)

## Tech Stack
- HTML5
- Vanilla CSS3
- CSS Reset for cross-browser consistency

## Development Workflow
Each feature is developed in a separate branch and merged to main via pull requests:
- `feature/basic-structure` - Project foundation and reset.css
- `feature/navigation` - Header and navigation menu
- `feature/hero` - Hero section with CTA
- `feature/services` - Services/features section
- `feature/showcase` - PC build showcase gallery
- `feature/about` - About/benefits section
- `feature/contact` - Contact form and footer

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!

## Deployment
The site is deployed using GitHub Pages
